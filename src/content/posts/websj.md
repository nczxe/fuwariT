# 工具类

## BuildCores
这是一个在线让你组装电脑的网站，让你熟悉组装电脑的步骤
[BuildCores](https://buildcores.com/)

## 吾爱破解
这是一个论坛，里面有很多有用的工具和软件以及病毒分析等
[吾爱破解](https://www.52pojie.cn/)

## 拍照模拟PhotoSkop
这是一个在线的拍照模拟软件，小白零成本试炼场
[PhotoSkop](https://www.photoskop.com/)

# 远程协助

## UU远程
由网易开发的串流游戏工具，但上面也能进行远程协助
[UU远程](https://uuyc.163.com/)

# 开发助力

## Qoder
Qoder是一个基于AI的代码生成器，您可以在上面输入自然语言描述，它会自动生成对应的代码
[Qoder](https://qoder.com/)

## TRAE
TRAE是字节跳动开发的一个AI IDE，高效助力开发
[TRAE](https://trae.cn/)

# 网页全收集

## 互联网档案馆
互联网档案馆收录了大量历史上的网页，您甚至可以在上面观看早已无法找到的旧电影！
[互联网档案馆](https://web.archive.org/)

## Fmhy
Fmhy是免费资源分享网站，它上面能找到很多免费资源
[Fmhy](https://fmhy.net/)

# 网页托管

## 赛博大善人Cloudflare
Cloudflare是全球有名的网络基础设施提供商，全互联网50%以上的流量都经过Cloudflare
[Cloudflare](https://www.cloudflare.com/)

## edgeone（腾讯云）
腾讯云edgeone是一个基于云的平台，它能提供高速的网页访问服务(中国地区需要备案)
[edgeone](https://www.tencentcloud.com/)

## vercel（Vercel）
vercel是一个基于云的平台，它能帮助您部署和托管您的网页应用程序
[vercel](https://vercel.com/)

# 代码托管库

## GitHub
GitHub是一个代码托管平台，上面有很多开源项目，本项目也托管在GitHub上，您可以在上面查看和下载代码
[GitHub](https://github.com/)

## huggingface
huggingface是一个AI模型托管平台，上面有很多常用的AI模型，您可以在上面查看和下载模型
[huggingface](https://huggingface.co/)
