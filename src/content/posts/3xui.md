## 3X-UI简介

3X-UI是一个由MHSanaei开发的节点自动搭建面板，只需在VPS上部署即可通过面板进行便捷的节点搭建。

::github{repo="MHSanaei/3x-ui"}

## 开始

1. 首先，我们需要一台VPS服务器，保证有公网IPv4/IPv6地址
2. 其次，一个域名，用于隐藏IP
3. 使用SSH连接的你的VPS服务器
4. 跟着下面操作

## 部署3X-UI

1. 我们首先先进行更新安装依赖

   ```bash
   apt update -y && apt install curl wget -y
   ```
  - 您可以在安装依赖完成后重启系统
    ```bash
     sudo reboot
     ```

2. 第二步，我们需要安装3X-UI。在SSH链接的窗口输入3X-UI安装脚本进行安装

   ```bash
   bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
   ```

3. 在cloudflare上添加DNS记录，指向你的VPS服务器的公网IPv4/IPv6地址
   - A记录：指向你的VPS服务器的公网IPv4地址
   - AAAA记录：指向你的VPS服务器的公网IPv6地址
   - 名称：你想设置的子域（如本站名称为blog，根域名为tncrr.us.kg，本站的网页就是blog.tncrr.us.kg）
   - 输入你的IP地址，将代理状态改为“仅DNS”，TTL设置为“自动”后保存
4. 接下来3X-UI会询问证书设置，这里选择输入“1”，使用Let's Encrypt证书，90天有效期，自动续订（请注意开放80端口，否则证书申请将会失败），这时3X-UI会让你填写你的域名，填写DNS记录中：”名称“.域名（这才是你DNS记录中指向你的VPS的域名）
5. 3X-UI会询问使用什么端口进行申请证书，直接回车（默认80端口）
6. 3X-UI会询问你端口开放，你可以自己选择面板端口，默认为随机端口。回复Y即可自行设置端口。
7. 自动续订证书以及其他设置，输入“Y”，选择第一个选项（输入”1“），3X-UI会询问你是否将证书传输到面板，输入“Y”
8. 我们将会得到Username（用户名），Password（密码），Port（端口），WebBasePath（面板路径），Access URL（面板访问URL）
9. （可选内容）在SSH窗口中输入x-ui进入面板终端控制器，选择“6”进行更改用户名与密码，它会询问你是否更改，输入”Y“
10. 开启BBR 输入x-ui，选择“24”，再选择”1“启用BBR，在显示BBR已启用后输入”0“回到菜单，输入”0”退出终端控制器
11. 访问面板给你Access URL，输入你的用户名与密码登录面板
12. 打开入站列表，添加入站，保持协议为VLESS不要动，端口改为443，安全中打开Reality，3X-ui已经帮你填写了大部分信息了，你只需要点击下方的“Get New Cert”与“Get New Key”，然后回到上方用户设置里将IFOW改为Xtls-rprx-vision即可
13. 点击创建，点击菜单>导出链接-订阅设置，复制到浏览器访问，即可获得订阅链接或者快速导入
14. 如果你对安全性有要求，你可以在面板设置URL设置中更改订阅路径。对网络安全有要求可以在Xray设置中启用并配置内置DNS服务器，以防止DNS劫持。
